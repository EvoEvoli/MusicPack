# MusicPack

This mod replaces the original music with my selection of soundtracks.
Contains music from : 
- Super Mario Galaxy
- Celeste
- Camellia
- Genshin Impact
- Chrono Trigger
- Star Rail
- Pokemon
- Terraria Calamity
- Zelda
- Undertale
And much more !

## Other Info

This mod was made with the base of the mod made by Contra : OriginalSoundTrack
